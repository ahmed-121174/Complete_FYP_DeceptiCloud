USAGE INSTRUCTIONS
================================================================================

1. TO USE THE MODEL:
   - Load: models/web_attack_detector_v2.keras
   - Feature Extractor: code/feature_engineering.py
   - Threshold: models/web_attack_threshold_corrected.txt (0.5)

2. TO RETRAIN:
   - Script: code/train_web_attack_v2.py
   - Requires: Dataset in Datasets/SQLI-nosqli-XSS/

3. TO DEPLOY:
   - Follow: documentation/WEB_ATTACK_DEPLOYMENT_GUIDE.md
   - Reference: documentation/WEB_ATTACK_FINAL_REPORT.md

4. QUICK START:
   ```python
   from code.feature_engineering import WebAttackFeatureExtractor
   import tensorflow as tf
   
   model = tf.keras.models.load_model('models/web_attack_detector_v2.keras')
   extractor = WebAttackFeatureExtractor()
   
   query = "SELECT * FROM users WHERE id = 1"
   features = extractor.extract_features(query)
   prediction = model.predict([list(features.values())])[0][0]
   
   is_attack = prediction >= 0.5
   print(f"Result: {'ATTACK' if is_attack else 'BENIGN'} ({prediction:.2%})")
   ```
